# Winged: Lost in Neon

# Trailer available at https://youtu.be/EosXg68gepI

# Game prototype developed for the Game Development Methodology course at IST
# 3D Competitive Racing Game (local co-op)
# Developed by Pedro Monteiro, Pedro Correia, José Maié, Rafael Martins and Francisca Mendonça
# 1 Semester of development with 2 weeks of coding
# Code in current build developed by Pedro Monteiro
# Level 2 not available

# To build and play:
# Install unreal engine
# Open Winged.uproject
# Add following assets from unreal marketplace to project:
# 	Paragon Fey (kora and kayn, fairies)
# 	Paragon Crunch (robot)
#	Paragon Shinbi (animation blueprint)
# 	Quadraped Creatures (spirit)
# To create executable: File -> Package project -> select platform -> choose folder
# First packaging may take a long time to compile shaders. Following builds are almost instant however.
# To play: run executable created in chosen folder

# For multiplayer use steam mode or ds4windows (current input settings for dualshock)
